# Ecommarce Site

This project was created e-commarce site with node js

## Installation for React (Frontend)
cd node_code
```bash
npm install
```

## Run frontend
```bash
npm start
```
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Installation for Node js (Backend)
cd server

### install Nodemon
```bash
npm install nodemon
```

### Run
```bash
npm run start
```
